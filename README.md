# 🌦️ SkyPulse Weather App

SkyPulse is a modern weather application that provides real-time weather updates, hourly forecasts, 7-day forecasts, AQI monitoring, and location-based weather detection.

---

## ✨ Features

- 🌍 Search weather by city
- 📍 Use current location weather
- 🌡️ Celsius / Fahrenheit toggle
- ⏰ 24-hour weather forecast
- 📅 7-day weather forecast
- 💨 Air Quality Index (AQI)
- 🌅 Sunrise & Sunset timings
- 🌬️ Wind speed & direction
- 💧 Humidity, pressure & visibility
- 🎨 Dynamic animated weather UI
- ⚡ Fast backend proxy with Express

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript

### Backend
- Node.js
- Express.js

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/skypulse-weather-app.git
```

### 2️⃣ Navigate into project folder

```bash
cd skypulse-weather-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the backend server

```bash
npm start
```

Server runs at:

```bash
http://localhost:3001
```

### 5️⃣ Open frontend

Open `index.html` in your browser.
