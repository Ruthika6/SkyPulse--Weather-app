# SkyPulse--Weather-app
SkyPulse provides real-time weather updates, hourly forecasts, 7-day forecasts, AQI monitoring, and location-based weather detection
✨ Features
🌍 Search weather by city
📍 Use current location weather
🌡️ Celsius / Fahrenheit toggle
⏰ 24-hour weather forecast
📅 7-day weather forecast
💨 Air Quality Index (AQI)
🌅 Sunrise & Sunset timings
🌬️ Wind speed & direction
💧 Humidity, pressure & visibility
🎨 Dynamic animated weather UI
⚡ Fast backend proxy with Express
🛠️ Tech Stack
Frontend
HTML5
CSS3
Vanilla JavaScript
Backend
Node.js
Express.js

Open-Meteo API

🚀 Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/stratos-weather-app.git
2️⃣ Navigate into project folder
cd stratos-weather-app
3️⃣ Install dependencies
npm install
4️⃣ Start the backend server
npm start

Server runs at:

http://localhost:3001
5️⃣ Open frontend

Simply open index.html in your browser.
